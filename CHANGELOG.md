# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-07

### Added
- Initial release
- Sonarr library analyzer with TV show analysis
- Radarr library analyzer with movie analysis
- Matching color schemes (Sonarr blue, Radarr orange)
- Sort options: Size, Per Episode/MB per Min, Name, Year, Codec, Resolution
- Filter/search functionality
- CSV export
- JSON export
- Codec detection (HEVC/AV1/H.264)
- Bit depth detection (8-bit vs 10-bit)
- Cookie persistence for credentials
- Responsive design
- Landing page with tool selection
